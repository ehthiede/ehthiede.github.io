About This Website

You might be wondering why this website looks like it was slapped together using rudimentary HTML. This is because it was slapped together using only rudimentary HTML. In fact, most of it is only half-remembered from a Computer Science A/B course taken a decade ago at the North Carolina School of Science and Mathematics.

I recognize that it is the 2019 and that people have made templates for pretty websites that use fancy web development tools named after coffee or jazz musicians. However, I like doing things myself, so I'm building it on my own. I expect the website to improve rapidly once I finish the w3schools html tutorial and learn things like "CSS".
